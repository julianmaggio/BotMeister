Bot Commands

Here are some of the available commands:

Command Description Example
/help Display the list of available commands. /help
/ping Check the bot's latency. /ping
/weatherstart Start the weather updates. /weatherstart
/weatherstop Stop the weather updates. /weatherstop
/Countdown [seconds] Set a countdown in seconds. /Countdown 60
/Clear [number] Clear a specified number of messages. /Clear 5
/userinfo Display information about the user. /userinfo
/serverinfo Display information about the server. /serverinfo
/roll [number] Roll a dice with the specified number of sides. /roll 6
